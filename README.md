ApkDetecter
===========

android apk查壳工具源代码

主要功能 :
----------
1. 检测DEX文件是否加固及加固厂商
2. 检测APK的基本信息：
  1. APKMD5值
  2. APK包名
  3. APK版本
  4. 签名信息等
3. DEX文件的字节信息

Translation
-----------

Tool for providing Android APK protector detection with UI

Main Features:
--------------
1. Detect DEX file protectors, obfuscators
2. Detect APK basic information
  1. APK MD5
  2. APK Package name
  3. APK version
  4. Signature informaton
3. Detailed information about the DEX file bytes
